## Tools

Any of these work:
- **Figma** (recommended) вҖ” precise control, export @2x
- **Canva** вҖ” use "Custom size 1280Г—640"
- **OG Image generators**: og-image.vercel.app, imgsrc.io

## How to upload (manual step вҖ” GitHub API does not support this)

```
1. Create image вҶ’ save as social-preview.png
2. GitHub вҶ’ Josepavese/aftertalk вҶ’ Settings (gear icon)
3. Scroll to "Social preview" section
4. Click "Edit" вҶ’ upload social-preview.png
5. Save
```
